# FramejsSample

frame.js を学習するためのサンプルリポジトリです。

## テンプレプロジェクトの作り方

```bash
yarn add frames.js
```

## 動かし方

- インストール

  ```bash
  cd my-project
  yarn
  ```

- フロントエンド起動

  ```bash
  yarn dev
  ```

## ローカルでデバックするためのツールをセットアップする方法

```bash
npm install -g @frames.js/debugger
```

以下コマンドで起動できる [http://localhost:3010](http://localhost:3010)

```bash
frames
```
